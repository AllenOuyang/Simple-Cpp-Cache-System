## 两个版本stable version & beta version
### beta version 使用方法
#### clone代码
```bash
git clone https://github.com/ArthurOyll/Simple-Cpp-Cache-System.git
```
#### 编译运行master
```bash
cd beta_version/Master/build
cmake ..
make
./master [端口号]
```
#### 编译运行 cacheserver
```bash
cd beta_version/CacheServer/build
cmake ..
make
./cacheserver
```
#### 编译运行client
```bash
cd beta_version/Client/build
cmake ..
make
./client
```
